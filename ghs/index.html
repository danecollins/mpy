AWR Browser Interface (ABI)

This page describes how to control AWRDE via a set of URL's which can be used from any web page such as guided help pages.

Many common tasks have direct URL commands but if you cannot accomplish a specific task you can use the ability to invoke an AWRDE script which gives you access to the entire AWRDE scripting API.

Overview

The ABI gets input in the form of standard URL's with arguments. The standard URL form is http://host/command?arg=value where

host
is the address of the server, localhost:8008 in this case
command
is the name of the command to be executed, OpenProject.py for example
arg
is the name of the argument if it exists
value
is the value for the argument, a file name for example

Each of the commands below will be in this format. For each command there is an explanation of the function of the command and a description of any arguments required. If a * appears next to an argument name that argument is required.

The command reference lists the command syntax for two distict content creation platforms. The Web syntax can be used in any HTML content creation platform. This syntax strives to be as simple as possible to embed into HTML documents

The Confluence Wiki syntax is used along with the awrcommand macro in confluence. This allows the command syntax to be much simpler to use. Since the macro supports multiple commands on multiple lines there is no need to combine multiple arguments into 1 statement.

For example, these are equivalent:

Web
http://localhost:8008/OpenSchematic?name=Resonator;close=1;tile=H
Confluence
CloseWindows
OpenSchematic Resonator
TileHorizontal

The commands are not alphabetical but rather in order of most common use

Command Reference

Open a project

This command loads a new project. If a project is already loaded it will be closed. The name of the project is the URL to the project. Currently Dropbox is used to store projects due to its convenience. The .vin file is also used.

The following arguments are supported:
name*
URL to the project to be downloaded and opened
simulate
set to 1 will cause the project to be simulated after it is opened
Web syntax http://localhost:8008/OpenProject?name=full_url_to_project.emp
Confluence syntax OpenProject full_url_to_project.emp

Open a schematic

This command opens a schematic window and then perform a tile command.

The following arguments are supported:
name
name of the schematic to be opened
close
a value of 0 does nothing, a value of 1 causes all existing windows to be closed
tile
a value of 'H' causes a horizontal tiling to be done, a value of 'V' causes a vertical tiling and a value of 'N' disables the tiling
Web syntax http://localhost:8008/OpenSchematic?name=ResonatorSchematic;close=1;tile=H
Confluence syntax OpenSchematic ResonatorSchematic

Simulate

This command run a simulation.

This command has no arguments
Web syntax http://localhost:8008/Simulate
Confluence syntax Simulate

Open Folder Contents

This command closes all open windows, opens all the contents of folder and then perform a tile command.

The following arguments are supported:
name
name of the folder to be opened
close
a value of 0 does nothing, a value of 1 causes all existing windows to be closed
tile
a value of 'H' causes a horizontal tiling to be done, a value of 'V' causes a vertical tiling and a value of 'N' disables the tiling
Web syntax http://localhost:8008/OpenFolder?name=AmpFolder;close=1;tile=H
Confluence syntax OpenFolder AmpFolder

CloseWindows

This command closes all open windows.

This command has no arguments
Web syntax http://localhost:8008/CloseWindows
Confluence syntax CloseWindows

Tile

This command tiles the open windows.

The following arguments are supported:
type
a value of 'H' causes a horizontal tiling to be done and a value of 'V' causes a vertical tiling
Web syntax http://localhost:8008/Tile?type=H
Confluence syntax Tile H

Execute VB Script

This command executes a Visual Basic script stored in the project.

The following arguments are supported:
name
name of the script to be executed
Web syntax http://localhost:8008/ExecuteScript?name=MyScript
Confluence syntax RunScript MyScript
